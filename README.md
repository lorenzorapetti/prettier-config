# `@lorenzorapetti/prettier-config`

> My personal [Prettier](https://prettier.io) config.

## Usage

**Install**:

```bash
$ yarn add --dev @lorenzorapetti/prettier-config
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "@lorenzorapetti/prettier-config"
}
```
